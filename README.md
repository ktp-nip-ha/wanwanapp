# 🐕 動く犬を撫でるアニメーション

犬をクリックまたはホバーして撫でてあげるシンプルなウェブアプリです。4枚のアニメーション画像で犬の反応が変わります。

## 機能

- 🎨 シンプルで可愛らしいデザイン
- 📱 レスポンシブ対応
- ✨ スムーズなアニメーション
- 💕 犬の状態表示とメッセージ

## 使用方法

1. 犬をクリックまたはホバーして撫でます
2. 犬がアニメーションで反応します
3. 楽しんでください！

## 技術スタック

- HTML5
- CSS3
- Vanilla JavaScript
- SVG画像

## GitHub Pages で公開する方法

1. このリポジトリを GitHub にプッシュします
2. リポジトリの Settings → Pages で公開設定をします
3. Branch を `main` (またはデフォルトブランチ)、フォルダを `/ (root)` に設定
4. Save をクリック
5. サイトが公開されます！

## ファイル構成

```
doglove/
├── index.html          # メインのHTMLファイル
├── images/            # SVG画像のディレクトリ
│   ├── dog-1.svg      # 犬が立っている状態
│   ├── dog-2.svg      # 犬が頭を下げた状態
│   ├── dog-3.svg      # 犬が舐めている状態
│   └── dog-4.svg      # 犬が満足している状態
└── README.md          # このファイル
```
